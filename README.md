# Obsidian-Messenger

Obsidian Messenger — безопасный мессенджер с сквозным шифрованием (end-to-end encryption).
Все сообщения шифруются на устройстве отправителя и расшифровываются только на устройстве получателя. Сервер не видит содержимое сообщений и хранит их в базе данных только в зашифрованном виде.

⚡ Особенности
Сквозное шифрование сообщений.
Сообщения сохраняются в зашифрованном виде в базе данных.
Полное удаление диалогов для обоих участников.
Управление друзьями: добавление и удаление пользователей.
Автоматическое создание нового пользователя при первом входе.

🛠 Установка
Клонируйте репозиторий и установите зависимости:
npm install
Запустите сервер:
npm start

После запуска сервер создаст файл messenger.db (SQLite) для хранения всех данных.

Сервер работает на порту 63000.
Доступ с localhost или через доменное имя (IP напрямую не поддерживается корректно).

🚀 Использование
Откройте страницу:
http://localhost:63000

Авторизация:
Введите любой логин и пароль. Пользователь создается автоматически при первом входе.
В дальнейшем можно использовать эти данные для входа.

Диалоги:
Удаление сообщений: кнопка удаления в диалоге полностью удаляет переписку для обоих участников.
Добавление друзей: введите имя пользователя в поле поиска и отправьте запрос на добавление. После подтверждения можно общаться.
Удаление друзей: удалите пользователя из списка друзей.

🗄 Хранение данных
Все данные хранятся локально в файле messenger.db.
Сервер не имеет доступа к содержимому сообщений.
Удалённые сообщения невозможно восстановить.
